# Sprint 3 · Лендинг v3.3 — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Привести лендинг в соответствие с SPEC v3.3 — сменить порядок воронки (AboutTrainer наверх, Objections перед Booking, короткий FinalCta), применить «закон рамок ПУТЬ A», внедрить фирменный фон Hero (сетка + cyan-glow за курсором), переписать карту (4 ЖК) и Transformation (4-этапный путь), привести Booking к 6 дням без субботы, добавить адаптив ≤420px.

**Architecture:** Сохраняем серверный рендер по умолчанию; `'use client'` только у виджетов (Quiz/BookingSlot/Objections) и у нового маленького `HeroBackground` (его задача — слушать `pointermove` и писать CSS-переменные `--mx/--my` без `setState`, GPU-only). Data-слой (`lib/types.ts` + `lib/quiz-data.ts` + `lib/slots-data.ts`) — единый источник; рамки управляются классом `.card` исключительно у виджетов (закон ПУТЬ A §5.8). Аддитивно меняем типы: добавляем v3.3 поля, старые помечаем `// DEPRECATED — sprint 3`; в финале (Task 17) сносим устаревшее.

**Tech Stack:** Next.js 16 (App Router) · React 19 · TypeScript strict · Tailwind v4 + `@theme`/`@layer` · `lucide-react` · Vitest + Testing Library · Playwright · Yandex Metrika через `next/script`.

**Эталоны:**
- `SPEC-trener-u-doma-v3.3.md` — текст спеки.
- `landing-final.html` (корень репо) — эталон вёрстки/CSS/JS-логики.
- `docs/superpowers/specs/2026-06-01-v3.3-update-design.md` — дизайн-документ.

**Команды (на проверках ниже):**
- `npm test` — Vitest (unit/RTL).
- `npm run test:e2e` — Playwright.
- `npm run typecheck` — `tsc --noEmit`.
- `npm run lint` — ESLint.
- `npm run build` — Next build.

```tsx
'use client';

import {useEffect, useRef} from 'react';

/**
 * HeroBackground (SPEC v3.3 §5.9)
 * Два слоя: статичная сетка-клетка (.hero-grid) + cyan-сетка под маской-пятном (.hero-glow).
 * Пятно следует за курсором через CSS-vars --mx/--my (без setState, без reflow).
 * Только десктоп: на touch и при reduced-motion слой glow скрыт CSS-медиа.
 */
export function HeroBackground() {
    const glowRef = useRef<HTMLDivElement | null>(null);

    useEffect(() => {
        if (typeof window === 'undefined') return;
        // Подвешиваемся только на hover-устройствах
        const supportsHover = window.matchMedia('(hover: hover)').matches;
        if (!supportsHover) return;

        const glow = glowRef.current;
        const hero = glow?.closest('.hero') as HTMLElement | null;
        if (!glow || !hero) return;

        const onMove = (e: PointerEvent) => {
            const r = hero.getBoundingClientRect();
            glow.style.setProperty('--mx', `${e.clientX - r.left}px`);
            glow.style.setProperty('--my', `${e.clientY - r.top}px`);
        };

        hero.addEventListener('pointermove', onMove);
        return () => hero.removeEventListener('pointermove', onMove);
    }, []);

    return (
        <>
            <div className="hero-grid" aria-hidden="true"/>
            <div
                ref={glowRef}
                className="hero-glow"
                aria-hidden="true"
                style={{'--mx': '50%', '--my': '50%'} as React.CSSProperties}
            />
        </>
    );
}
```
