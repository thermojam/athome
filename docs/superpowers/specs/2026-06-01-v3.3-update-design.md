**Дата:** 2026-06-01
**Ветка:** `feat-spec-3.3`

## Цель спринта

Привести лендинг в соответствие с v3.3:
1. Сменить порядок воронки (AboutTrainer наверх; Objections перед Booking; короткий FinalCta).
2. Применить «закон рамок ПУТЬ A» (§5.8 спеки): рамки — только у виджетов, повествование без рамок.
3. Внедрить фирменный фон Hero (сетка + интерактивный cyan-glow за курсором на десктопе).
4. Переписать карту (4 ЖК, без рамки секции, залив снизу) и Transformation (4-этапный путь вместо BAB-колонок).
5. Привести Booking к v3.3 (6 дней без субботы, без тегов hint/walk, кнопка auto-width).
6. Добавить адаптив ≤420px (320/360/390/414).

**Главная метрика спринта:** визуальный gate (§14 спеки) и «5 приёмов премиальности» (§5.2) — пройдены на каждой секции. Поведенческих изменений в квизе/слоте/аналитике нет.

## Подход (организация работы)

**Approach B** — стадии в одной ветке `feat-spec-3.3`, итоговый PR один. Стадии:

1. Data layer (types, CONTENT, BRIDGES, slots).
2. Tokens + base CSS (globals.css).
3. Новый `AboutTrainer`.
4. Рефакторинг существующих секций (Hero, Problem, LocationMap, Transformation, BookingSlot, FinalCta) — по одной за коммит.
5. `app/page.tsx` — новый порядок + bridges + e2e порядка секций.
6. Финал: lint/tsc/vitest/playwright/build все зелёные, визуальный gate.

TDD на каждой стадии: RED (новые/обновлённые тесты падают) → GREEN (код) → REFACTOR.

## 10. Риски и митигация

| Риск | Митигация |
|---|---|
| Hero glow дёргается на слабых машинах | Только запись CSS-vars (нет setState), нет reflow. `prefers-reduced-motion` отключает. |
| Map SVG ломается под Tailwind purge | Все цвета/glow в SVG — inline-стили или через CSS-vars, не utility-классы. |
| Aspect-ratio 16/9 → 1/1 на ≤420 может разъехать пины | Координаты пинов в % от bounding box, не зависят от aspect — проверено в эталоне. |
| Замена `before/after` → `steps` ломает существующие `Transformation` тесты | Старые тесты переписываются в Stage 4. Секция-якорь `#bab` сохраняется. |
| Фото отсутствует к моменту визуального gate | Fallback `.about-photo-placeholder` (градиент violet→cyan + бейдж) — секция выглядит цельно без блокировки спринта. |
| `lead_click_direct` теряет одну точку запуска (Hero) | Сохраняется в About и Final — две точки клика на TG остаются; общая воронка эквивалентна. |
